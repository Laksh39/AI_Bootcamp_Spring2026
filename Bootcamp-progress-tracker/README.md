# 🚀 AI Bootcamp Progress Tracker

A gamified progress tracker built to log and visualize my journey through the 15-Day AI Pro-Builder Bootcamp (Spring 2026). Every mission completed earns XP, every tool used gets logged, and every level up feels earned.

**Live demo → [laksh39.github.io/New-Bootcamp-Progress-Tracker](https://laksh39.github.io/New-Bootcamp-Progress-Tracker/)**

---

## ✨ Features

**🎮 XP & Leveling System**
Complete missions to earn XP and level up. Progress is tracked visually so you always know how far you've come and how far you have to go.

**📋 15 Mission Cards**
One card per day of the bootcamp. Each mission can be marked complete, with tools used logged per session.

**🔒 Parent Authentication**
A SHA-256 hashed password system that requires parent authorization before progress can be modified — built with accountability in mind.

**☁️ Firebase Realtime Database**
Progress syncs live to Firebase so data persists across devices and sessions. No progress is ever lost.

No installation needed — just open the live link above. If self-hosting, clone the repo and update the Firebase config in `index.html` with your own project credentials.

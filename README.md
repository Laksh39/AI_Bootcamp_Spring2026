## The Story

I didn't come into this bootcamp knowing how to deploy a web app, run a language model locally, or wire up a voice pipeline on a Raspberry Pi. By Day 15, I had done all three.

This bootcamp was 15 days of waking up, picking up a new AI tool, and not stopping until something actually worked. Some days that meant debugging Firebase at midnight. Some days it meant watching a Raspberry Pi slowly process a sentence and feeling genuinely amazed when it talked back.

What got me into AI wasn't just the hype — it was the idea that machine learning could actually *understand* things. That a model running on a tiny Pi sitting on my desk could listen to my voice, think, and respond. That's the thing that hooked me, and that's what most of these projects are about: **AI that runs at the edge, close to the real world, not just in the cloud.**

## ⭐ Featured Projects

### 🕵️ Boss Detector *(Day 10)*
A computer vision system running entirely on a **Raspberry Pi 5** that detects when a person walks into frame using a live camera feed. No cloud. No API calls. Just a Pi, a camera, and a local vision model via Ollama doing real-time inference.

I built this because I wanted to prove that meaningful AI doesn't need a data center behind it. A $80 piece of hardware can see the world — and that still blows my mind a little.

**Key highlights:**
- Fully on-device — no data leaves the Pi
- Real-time person detection via local AI inference
- Built and deployed in a single day

---

### 🎙️ Voice Assistant *(Day 13)*
This was the one I was most proud of. A fully working voice assistant on the Raspberry Pi 5 — you speak, it listens, thinks, and talks back. The whole pipeline: speech-to-text → local LLM via Ollama → text-to-speech, running entirely on the edge.

The first time it actually responded to my voice with a coherent answer, I just sat there for a second. It felt like science fiction running on my desk.

**Key highlights:**
- Complete voice pipeline with no internet dependency
- Runs locally on Raspberry Pi 5
- Real-world edge-AI deployment, not just a demo
